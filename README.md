## Overview

This project is a temperature converter web application that allows users to seamlessly convert temperatures between Celsius and Fahrenheit. The application provides real-time conversion as users type, making it quick and intuitive to use.

## Features

- **Real-time Conversion**: Instant temperature conversion as you type
- **Bidirectional**: Convert from Celsius to Fahrenheit or vice versa
- **Clean Interface**: Simple, modern design with gradient background
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Quick Reference**: Helpful temperature reference points (freezing, boiling, etc.)
- **Accessible**: Built with semantic HTML and ARIA labels
- **Clear Function**: One-click button to reset all inputs
